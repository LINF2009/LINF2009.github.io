函数

函数的概念

对某一个特定的功能或者代码块进行封装，在需要使用该功能时直接调用即可。

函数是组织好的，可重复使用的，用来实现单一，或相关联功能的代码段。

函数能提高应用的模块性，和代码的重复利用率。你已经知道Python提供了许多内建函数，比如print()。但你也可以自己创建函数，这被叫做用户自定义函数。

定义一个函数

你可以定义一个由自己想要功能的函数，以下是简单的规则：

  • 函数代码块以 def 关键词开头，后接函数标识符名称和圆括号 **()**。
  • 任何传入参数和自变量必须放在圆括号中间，圆括号之间可以用于定义参数。
  • 函数的第一行语句可以选择性地使用文档字符串—用于存放函数说明。
  • 函数内容以冒号 : 起始，并且缩进。
  • return [表达式] 结束函数，选择性地返回一个值给调用方，不带表达式的 return 相当于返回 None。

语法：

定义函数时也可以在参数列表中加入参数。

参数：在函数调用时给函数传递一些信息。

分类：

1.形参，在函数定义的时候，需要准备一些变量来接收信息

2.实参，实际在调用的时候传递的信息

形参就相当于是上图中的width和height，而实参就相当于是上图的a和b（在area（）函数中的参数）。

实参分类：

  • 位置参数：按照位置传递参数，就像上图的实参a,b。

  • 关键字参数：按照参数的名字进行传递参数。

  • 混合参数：

    顺序：位置参数放前面，关键字参数放后面，否则报错

注意：在实参执行的时候，必须要保证形参有数据。

形参分类：

  1. 位置参数：按照位置一个一个去声明变量。
  2. 默认值参数：在函数声明时给变量一个默认值，若实参没有值则默认值生效，否则就不生效。

  1. 动态传参：
    • *args，表示接收所有位置参数的动态传参。（接收到的参数放入元组中）

  • **kwargs，接收所有关键字的动态传参。（接受道德参数放入字典中）

顺序：位置参数 > *args > 默认值参数 > **kwargs

若是默认值参数在*args前面则默认值参数只用在关键字参数后面没有实参时才能生效，所以*args必须要在默认值参数前面。

假设有一个列表和一个接收所有值的函数，需要将所有列表中的值传入函数中可以将函数（*+列表）作为实参，可以直接将列表的值都传入函数中。

函数返回值

函数调用执行后，会给调用方 一个结果，这个结果解释返回值。

若返回值为空或没有写return，则函数返回值为None。

无论return下面有多少条语句，函数都不会执行下去。

若return后面不跟数据，此时接收到的返回值依然是None。

return+值此时外界就能收到一个return返回的值。

若return后面有多个值则外界就会收到一个元组。

变量的作用域

上图中的a就是全局变量，b是局部变量。

在函数中也可以调用全局变量，而在函数外无法调用局部变量。

函数的作用域是全局。

函数的嵌套

局部变量函数在外面无法访问，只有在func1（）中可以调用func2（）。

在func1（）设置返回值为func1（）中的函数，并赋值给一个变量，此时该变量就可以执行函数了。 所以函数可以相当于一个变量。都表示一个内存地址。

注意：实参也可以是一个函数。将inner（）作为一个实参传入func1（）中，在func1（）中执行inner（）函数。

两个关键字

若声明一个全局变量a = 10,再声明一个函数，在函数中再声明一个变量a = 20,在函数体外面执行函数并打印变量a。

此时函数中的a是函数创建的一个全局变量，没有改变全局变量的a。

注意：nonlocal关键字是向函数外找一层，看有没有相应的变量，如果有则引入，没有则报错（不包括全局变量）。

闭包

定义一个函数并声明一个变量，再定义一个嵌套函数该函数引入了外层函数的变量并且返回值为该变量，第一个函数的返回值是嵌套函数名。

本质是内层函数对外层函数的局部变量的使用，此时内层函数被称为闭包函数。

由于需要闭包函数能一直使用所以闭包的变量需要一直存在。

由于再函数外无法修改函数内的局部变量，只有使用闭包函数才能修改，所以将全局变量放入闭包函数中可以可以避免全局变量被修改。

作用：

1.可以让一个变量常驻于内存。

2.可以避免全局变量被修改。

装饰器

1.函数可以作为参数进行传递。

2.函数可以作为返回值进行返回。

3.函数名可以像变量一样进行赋值操作。

结论：装饰器本质上是一个闭包，作用是在不改变原有函数调用的情况下，给函数增加新的功能。

传入函数fn和返回函数inner不能加括号。

装饰器常用于用户自动登录，日志。
